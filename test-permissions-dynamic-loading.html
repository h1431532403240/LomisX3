🔐 權限動態載入測試
LomisX3 AuthStore V6.6 - 驗證動態權限載入機制

📡 API 端點要求

GET /api/auth/permissions

回應格式： { data: { permissions: string[] } }

認證： Bearer Token (自動從 localStorage 讀取)

說明： 後端實現此 API 後，前端將自動在登入、初始化、用戶資料恢復時載入最新權限

🚀 測試 1: 手動觸發權限載入
1. 確保已登入 LomisX3 系統並有有效的認證 token
2. 點擊「載入權限」按鈕測試動態權限載入
3. 觀察控制台日誌和權限狀態變化
🔑 測試 2: 登入流程權限載入
1. 模擬登入成功後的權限載入流程

3. 檢查權限載入的執行順序和錯誤處理
🛡️ 測試 3: 權限檢查功能

2. 驗證 PermissionGuard 組件使用新的權限系統

⚠️ 測試 4: 錯誤處理機制
